Quizzie

Quizzie is a scalable and cost‑efficient web‑based quiz platform designed to address the limitations of expensive commercial quiz‑conducting systems. The platform enables examiners to create and manage customizable assessments with advanced configuration options, seamless user experience, and integrated integrity monitoring features suitable for academic and public examination environments.

Problem Statement

Existing quiz and examination platforms often involve high subscription costs, limited customization, and insufficient integrity‑monitoring capabilities. Educational institutions and independent examiners require an affordable, flexible, and secure system that supports multiple question formats, configurable marking schemes, and reliable anti‑cheating mechanisms without compromising usability or scalability.

Existing Solutions Analysis

Review of commercial quiz platforms and lightweight academic systems highlighting cost barriers, restricted configuration control, and limited monitoring support during examinations.

Proposed Contribution

Development of a customizable, low‑cost, and secure quiz platform featuring flexible exam creation, support for single‑correct and multiple‑correct MCQs, configurable marking logic, and integrated integrity monitoring using camera access, microphone activity checks, and automated flagging mechanisms.

System Methodology

Data & Exam Configuration

Structured exam creation supporting multiple question types, configurable marks, validation rules, and efficient storage for scalable assessment management.

System Architecture

Modular full‑stack web architecture ensuring responsive interface design, secure communication, role‑based access, and maintainable backend logic for reliable deployment.

Integrity Monitoring

Anti‑cheating measures including camera access, microphone activity detection, and automated behaviour flagging to support fair and transparent examinations.

Results & Analysis

Operational Efficiency 92%

Quizzie demonstrates improved affordability, configurability, and examination integrity compared to traditional quiz‑conducting platforms, while maintaining a streamlined and user‑friendly assessment experience.
